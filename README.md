# AI网站导航平台

## 网站概述
收集多家公司旗下的AI产品，提供相关链接、介绍或使用教程，帮助用户快速找到所需的AI资源。

##  核心功能
- **网站搜索**：支持按关键词搜索AI网站
- **分类浏览**：按公司分类浏览AI网站
- **用户认证**：支持用户注册、登录和登出
- **网站管理**：支持添加、编辑、删除网站信息
- **新闻展示**：展示最新的AI资讯
- **响应式设计**：适配不同屏幕尺寸

##  技术栈
- **前端技术**：HTML5 + CSS3 + JavaScript (ES6+)
- **数据存储**：LocalStorage 本地数据存储
- **布局方案**：Flexbox/Grid 响应式布局

##  内容来源
- 知乎
- B站
- 各AI公司官方网站

##  参考网站

##  站点组成页面
1. **首页**：展示最新AI资讯和搜索功能
2. **网站列表页**：按公司分类展示AI网站
3. **网站详情页**：展示网站详细信息
4. **添加/编辑页**：管理网站信息
5. **帮助页**：使用指南和技术说明
6. **登录/注册页**：用户认证
