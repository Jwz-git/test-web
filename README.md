# 站点简介

本站点是一个AI产品导航平台，收集并展示多家公司旗下的AI产品，提供相关链接、介绍和使用教程，帮助用户快速找到所需的AI资源。

## 核心功能

- **AI产品展示**：收集并分类展示各类AI产品
- **产品详情**：提供详细的产品介绍和使用信息
- **搜索筛选**：支持按关键词搜索和分类筛选
- **添加产品**：允许用户提交新的AI产品信息
- **用户认证**：提供用户注册和登录功能
- **响应式设计**：适配不同设备屏幕

## 技术栈

- 前端：HTML5、CSS3、JavaScript
- 样式预处理器：SCSS
- 构建工具：npm scripts

# 使用说明

## 快速开始

1. **访问站点**
   [网页连接](https://jwz-git.github.io/test-web/)

2. **浏览AI产品**
   - 在首页查看推荐的AI产品
   - 点击"产品列表"查看所有AI产品
   - 使用搜索框搜索特定产品

3. **查看产品详情**
   - 点击任意AI产品卡片，进入产品详情页
   - 查看产品介绍、功能特点和使用教程

4. **添加新产品**
   - 点击"添加网站"按钮
   - 填写产品信息并提交
   - 注册用户可以登录后进行操作

5. **注册/登录**
   - 点击"登录/注册"按钮
   - 新用户：填写注册信息完成注册
   - 老用户：直接输入账号密码登录

6. **获取帮助**
   - 点击"帮助"按钮
   - 查看使用指南和常见问题

## 详细功能说明

### 首页 (index.html)
- **AI产品推荐**：展示精选的AI产品
- **快速搜索**：支持关键词搜索
- **导航菜单**：快速访问其他页面

### 列表页 (list.html)
- **产品分类展示**：按类别展示AI产品
- **筛选功能**：支持按条件筛选

### 详情页 (web_detail.html)
- **AI信息**：详细介绍AI产品
- **API调用**：提供简单API调用

### 添加AI页 (form.html)
- **AI表单**：填写产品名称、描述、链接等信息
- **提交功能**：将新产品添加到数据库

### 登录注册页 (login_register.html)
- **用户注册**：创建新账号
- **用户登录**：已有账号登录

### 帮助页 (help.html)
- **使用指南**：详细的站点使用说明

## 开发使用

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm start
```
服务器将在 http://localhost:3000 启动

### 构建项目
```bash
npm run build
```
